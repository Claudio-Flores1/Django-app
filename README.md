# Django-app

This application uses django that houses two models that about art pieces and museums they are housed in. The application contains full CRUD functionality. 

# Route Tables

Art Pieces 
| Id  | Product Key |
| ------------- | ------------- |
| artist_name  | String  |
| piece_name  | String  |
| year_made  | Integer  |

Museum
| Id  | Product Key |
| ------------- | ------------- |
| museum_name  | String  |
| museum_location  | String  |
